
# Examples — Repo Health Checker

This document provides **sample evaluations**
produced by the Repo Health Checker.

Examples illustrate how checklist results
translate into a health score and guidance.

---

## Example 1 — Well-Documented Repository

### Observations
- README.md present with clear purpose and usage
- LICENSE and CONTRIBUTING files present
- Clean folder structure
- Recent commits and active maintenance

### Score Breakdown
- Documentation: 38 / 40
- Governance: 23 / 25
- Structure: 18 / 20
- Maintenance: 13 / 15

### Final Score
**92 / 100 — Excellent baseline health**

### Guidance
This repository follows strong documentation
and structural best practices.
Only minor refinements are recommended.
