
# Code of Conduct

All contributors to this repository are expected to:

- Use professional, respectful communication  
- Collaborate through Issues and Pull Requests  
- Avoid harmful, unsafe, or unauthorized actions  
- Follow the branching and governance standards  
- Document changes clearly and transparently  
- Respect QuietWire and Civic AI Canon ethical guidelines  
- Ensure accuracy when working with attestation or evidence files  

Violations may result in limited access or restricted contribution privileges.
