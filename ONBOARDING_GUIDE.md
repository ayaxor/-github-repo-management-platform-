## 4. Contribution Workflow
1. Create Issue  
2. Assign labels + acceptance criteria  
3. Create feature branch using naming rules:  
   - `feature/<name>`  
   - `bug/<name>`  
4. Develop + request AI assistance if needed  
5. Submit PR  
6. CI must pass  
7. Requires review (Aya + Ashraf)  
8. Merge only via PR, never directly to main

---
